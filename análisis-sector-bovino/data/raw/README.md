# Datos en Bruto (Raw Data)

Esta carpeta contiene las bases de datos originales utilizadas en el Trabajo de Fin de Grado (TFG) para el análisis multivariante del sector bovino español.

## Contenido Esperado

- `BASE_1_ESTRUCTURAL.xlsx`: Datos estructurales y demográficos de las explotaciones bovinas por comunidad autónoma.
- `BASE_2_ECONOMICA.xlsx`: Datos económicos y financieros del sector (producción, precios, subvenciones).
- `BASE_3_SOSTENIBILIDAD.xlsx`: Indicadores de sostenibilidad y presión ganadera.

*Nota: Los archivos de datos `.xlsx` se encuentran ignorados por control de versiones según las reglas del archivo `.gitignore` para evitar subir archivos pesados, pero esta carpeta y este archivo de documentación son rastreados.*
